# Herbarium.Rainfall.Fitness

Authors: Megan Bontrager, Samantha J. Worthy, Laura Leventhal, Julin N. Maloof, Jennifer R. Gremer, Johanna Schmitt, Sharon Y. Strauss

Accepted for Publication in _New Phytologist_
